# Materia Demo

Sets up up a virter VM for a quick Materia demo.

1. Run `mise setup-vm`
2. SSH into VM with `virter vm ssh materia-demo`
3. Create materia directories and grab materia binary with `sh /root/install_materia.sh`
4. Setup materia config with `source /root/materia.env`
5. Run materia commands as desired.
